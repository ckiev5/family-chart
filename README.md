## Features

- üå≥ **Interactive Family Trees** - Zoom, pan, and navigate through complex family structures
- üé® **Customizable Styling** - Full control over colors, fonts, and layout
- ‚ö° **Framework Agnostic** - Works with React, Vue, Angular, Svelte, or vanilla JavaScript
- üéØ **TypeScript Support** - Full type definitions included
- üìä **D3.js Powered** - Built on the powerful D3.js visualization library
- üéØ **Easy to Use** - Simple API with many examples
- üîÑ **Real-time Updates** - Dynamic tree updates and modifications
- üé® **Multiple Card Types** - SVG and HTML card components
